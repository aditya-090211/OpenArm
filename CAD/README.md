the link to the document is [here](https://cad.onshape.com/documents/df36195e73c08f9a76f4d7ba/w/6e8ef89e1a75c97e46a746ab/e/d52665107da01e4d7b02bdb3?renderMode=0&uiState=6a09e84c662c2d04bcc73d98)
The full OpenArm assembly is designed and maintained in Onshape.

Instead of uploading individual part files to this repository, the complete Onshape document is linked directly. Since the design is still actively evolving and parts are frequently updated, maintaining a single centralized CAD source helps ensure that all files, assemblies, and revisions remain consistent and up to date.

This also allows anyone viewing the project to:

* access the latest version of the CAD
* inspect assemblies and part relationships
* download individual components
* export parts in their preferred file format

The goal is to keep the CAD workflow as open, accessible, and maintainable as possible throughout development.
