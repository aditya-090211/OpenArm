# OpenArm

## Open-Source Affordable Robotic Arm Platform

OpenArm is an open-source robotics project focused on developing a low-cost, modular, and accessible robotic arm platform for students, makers, researchers, and schools around the world.

The goal of this project is to reduce the cost and complexity of robotic arm systems while improving accessibility to hands-on robotics education and engineering.

---

# Vision

Many existing educational robotic arms are:
- expensive
- difficult to assemble
- mechanically complex
- inaccessible to many students and schools

OpenArm aims to create a robotic arm platform that is:
- affordable
- modular
- lightweight
- repairable
- open-source
- easy to manufacture and assemble

The long-term goal is to help make robotics education more accessible globally, especially in communities where access to engineering resources is limited by cost.

---

# Project Background

This project began in 2023 with extremely simple robotic arm prototypes made using:
- candy sticks
- cardboard
- micro servos
- Arduino

Over time, the project evolved into increasingly advanced CAD-based robotic arm systems designed in Onshape, incorporating:
- gear-driven joints
- modular arm structures
- multi-axis movement
- gripper systems
- improved structural rigidity

Most of the CAD, robotics, and prototyping knowledge used throughout this project has been self-taught through experimentation, iteration, and participation in competitive robotics.
