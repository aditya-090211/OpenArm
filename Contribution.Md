## Contributing

Contributions, feedback, and suggestions are always welcome.

If you would like to contribute to the project through:
- CAD improvements
- control systems
- software development
- documentation
- testing
- educational resources
- manufacturing ideas

please feel free to open an issue or submit a pull request.

## Sponsorship & Support

OpenArm is an independent open-source project focused on improving accessibility to robotics education.

If you are interested in supporting, sponsoring, or collaborating on the project — including helping expand access to underprivileged schools and students — please feel free to get in touch.
